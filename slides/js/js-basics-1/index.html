Тег script

Для вставки скрипта в документ мы просто пишем <script>, без дополнительных атрибутов и комментариев.

		<script>
			alert('Mango says woof!');
		</script>

Внешние скрипты

JavaScript код выносят в отдельный файл, который подключается в HTML.

		<script src="/path/to/script-1.js"></script>
		<script src="/path/to/script-2.js"></script>

Синхронные скрипты

Браузер загружает и отображает HTML постепенно. Если браузер видит тег <script>, то он по стандарту обязан сначала выполнить его, а потом показать оставшуюся часть страницы. Такое поведение называют «синхронным».

		<p>First paragraph.</p>
		<script>
			alert('Hello!');
		</script>
		<p>Second paragraph.</p>

Асинхронные скрипты

Атрибут async - cкрипт выполняется полностью асинхронно.
То есть, при обнаружении <script src="..." async> браузер не останавливает обработку страницы, а спокойно работает дальше. Когда скрипт будет загружен – он выполнится.

Атрибут defer - скрипт также выполняется асинхронно, не заставляет ждать страницу, но есть два отличия от async. 

		<script src="script-1.js" defer></script>
		<script src="script-2.js" defer></script>
		html markup
  • Браузер гарантирует, что относительный порядок скриптов с defer будет сохранён
  • Скрипт с defer сработает, когда весь HTML-документ будет обработан браузером

Современный стандарт

Чтобы перевести код в режим полного соответствия современному стандарту, нужно указать специальную директиву use strict.

		'use strict';
		...

Переменные

  • Переменные используются для хранения информации.
  • Переменная состоит из имени и выделенной области памяти, которая ему соответствует.
  • Для объявления переменной используется ключевое слово var: var message;
  • После объявления, можно записать в переменную данные: message = 'Hello';
  • Эти данные будут доступны при обращении по имени: console.log(message);

Имена переменных

  • Имя может состоять из: букв, цифр, символов $ и _
  • Первый символ не должен быть цифрой
    • не верно - var 123myVariable;
    • верно - var myVariable123;
  • Регистр букв имеет значение: var apple; и var aPplE; это две разные переменные
  • Зарезервированные имена: var, class, return, export ...
  • Имя переменной должно быть понятным
    • плохо - var yabloko, chislo, massivProductov, naborIz4;
    • хорошо - var apple, num, goodsArr, list;

Типы данных

  • Число «number»: var n = 10;
  • Строка «string»: var str = "just some text";
  • Булевый тип «boolean»: var status = true;status = false;
  • Специальное значение «undefined»: var age = undefined;
  • Специальное значение «null»: var name = null;
  • Объекты «object»: var dog = { age: 1, name: "Mango" };

Преобразование типов для примитивов

Три типа преобразования:

  • Строковое преобразование - происходит, когда требуется представление чего-либо в виде строки.
  • Числовое преобразование - происходит в математических функциях и выражениях, а также при сравнении данных различных типов (за исключением строгих сравнений).
  • Преобразование к логическому значению - происходит в логическом контексте, таком как if(value), и при применении логических операторов.

Основные операторы

  • Сложение: var a = 2 + 2; // 4
  • Вычитание: var b = 3 - 2; // 1
  • Умножение: var c = 3 * 3; // 9
  • Деление: var d = 4 / 2; // 2
  • Остаток от деления: var e = 5 % 2; // 1

Основные операторы

Конкатенация (сложение строк):

		var str = "I " + "love " + "Java" + "Script";
		console.log(str) //I love JavaScript

Приведение строки к числу:

		var a = 1, b = "2";
		a + Number(b) // 3
		a + +b // 3

Инкремент и декремент

Инкремент(постфиксный):

				var a = 2;
				console.log(a++); //2

Инкремент(префиксный):

				var a = 2;
				console.log(++a); //3

Декремент(постфиксный):

				var b = 5;
				console.log(b--); //5

Декремент(префиксный):

				var b = 5;
				console.log(--b); //4

Операторы сравнения

  • Больше/меньше: a > b, a < b
  • Больше/меньше или равно: a >= b, a <= b
  • Равенство: a == b
  • Неравенство: a != b
  • Строгое равенство: a === b
  • Строгое неравенство: a !== b

Логические(Булевые) значения

  • true – имеет смысл «да», «верно», «истина», 1
  • false – означает «нет», «неверно», «ложь», 0
		console.log( 5 > 2 ); // true, верно
		console.log( 3 == 1 ); // false, неверно
		console.log( 3 != 1 ); // true, верно
		console.log( 0 == false ); // true, верно
		console.log( 0 === false ); // false, неверно

Взаимодействие с пользователем

  • alert: alert(text);
  • prompt: var name = prompt(text, default);
  • confirm: var isHuman = confirm(text);

Условный оператор if

Когда в зависимости от условия нужно выполнить различные действия, используется оператор if.

		var age = prompt('Введите Ваш возраст: ', '');
		if (age < 18) {
			alert('младше 18');
		}

Неверное условие, else

Необязательный блок else («иначе») выполняется, если условие неверно.

		var age = prompt('Введите ваш возраст: ', '');
		if (age < 18) {
			alert('младше 18');
		} else {
			alert('старше или ровно 18');
		}

Несколько условий, else if

Если нужно проверить несколько вариантов условия используется блок else if.

		var age = prompt('Введите ваш возраст: ', '');
		if (age < 18) {
			alert('младше 18');
		} else if (age == 18) {
			alert('ровно 18');
		} else {
			alert('старше 18');
		}

Логические операторы

|| (OR) - вычисляет операнды слева направо до первого «истинного» и возвращает его, а если все ложные – то последнее значение.

		var number = 20;
		if(number < 5 || number > 15) {
			console.log('true');
		}

&& (AND) - вычисляет операнды слева направо до первого «ложного» и возвращает его, а если все истинные – то последнее значение.

		var name = 'Mango', isHappy = true;
		if(name === 'Mango' && isHappy) {
			console.log('woof!');
		}

! (NOT) - Сначала приводит аргумент к логическому типу true/false, затем возвращает противоположное значение.

console.log(!true); // false

Цикл while

Синтаксис:

				while(условие) {
					// код, тело цикла
					// код, тело цикла
					// код, тело цикла
				}

Пример:

				var i = 0;
				while(i <= 5) {
					console.log(i);
					i++;
				}

Цикл do while

Синтаксис:

				do {
					// код, тело цикла
					// код, тело цикла
					// код, тело цикла
				} while(условие)

Пример:

				var i = 0;
				do {
					console.log(i);
					i++;
				} while(i <= 5)

Цикл for

Синтаксис:

				for(инициализация; условие; итерация) {
					// код, тело цикла
					// код, тело цикла
				}

Пример:

				for(var i = 0; i < 5; i++) {
					console.log(i);
				}

Цикл выполняется так:

  • Инициализация - выполняется один-единственный раз, при заходе в цикл.
  • Условие - проверяется перед каждой итерацией и при входе в цикл, если оно нарушено, то происходит выход.
  • Тело - выполняется в случае удовлетворения условия.
  • Итерация - выполняется после тела на каждой итерации, но перед проверкой условия.

Строки

В JavaScript любые текстовые данные это строки.

var str = 'This is a string';

Свойства и методы

  • str.length - вернет длину строки
  • str.charAt(n) - вернет строку из одного символа на позиции n
  • str.toLowerCase() и str.toUpperCase() - вернут строку в соответствующем регистре
  • str.indexOf(val) - вернет позицию на которой находится подстрока или -1, если ничего не найдено

Массивы

Массив – разновидность объекта, где в качестве ключей выбраны цифры, с дополнительными методами и свойством length. var arr = [1, 2, '3', '4'];

Свойства и методы

  • arr.length - вернет длину массива.
  • arr.pop() - удаляет последний елемент массива и возвращает его.
  • arr.push(val) - добавляет элемент в конец массива.
  • arr.shift() - удаляет из массива первый элемент и возвращает его.
  • arr.unshift(val) - добавляет элемент в начало массива.
Обязательно к прочтению

Массивы: методы

  • split(val, count) - - превратит строку в массив, разбив ее по разделителю val.

    var arr = 'hello'.split(''); console.log(arr); // ['h', 'e', 'l', 'l', 'o']

  • join(val) - берет массив и склеивает его в строку, используя val как разделитель.

    var str = arr.join(''); console.log(str); // hello

  • splice(index[, deleteCount, elem1, ..., elemN]) - начиная с позиции index, удалить deleteCount элементов.

    arr.splice(1, 1);console.log(arr); // ['h', 'l', 'l', 'o']

  • slice(begin, end) - копирует элементы от begin, до end (не включая end)

    var newArr = arr.slice(1, 3); console.log(newArr); // ['l', 'l']

Функции

Функции это один из фундаментальных строительных блоков в JavaScript. Функция это JavaScript процедура - набор операторов, который выполняет задачу или вычисляет значение.

			function имя функции(список параметров) {
			...тело функции...
			}
			...
			имя функции(...); //вызываем функцию по имени

Функции: область видимости переменных

В JavaScript область видимости переменных ограничена телом функции.

		function showMessage() {
			var message = 'Mango says woof!'; // локальная переменная
			alert(message);
		}
		...
		showMessage(); // 'Mango says woof!'
		...
		alert(message); // <-- будет ошибка, т.к. переменная видна только внутри showMessage

Функции: возврат значения

Функция может возвратить результат, который будет передан в вызвавший её код.

  • Для возврата значения используется директива return
  • return может находиться в любом месте функции. Как только до return доходит управление – функция завершается и значение передается обратно.
  • Вызовов return может быть и несколько.
  • Директива return может также использоваться без значения, чтобы прекратить выполнение и выйти из функции. 
			function foo(a, b) {
				return a + b;
			}
			console.log(foo(2, 3)); // 5
Обязательно к прочтению

Функциональные выражения

В JavaScript функция является значением, таким же как строка или число.

  • Function Declaration – функция, объявленная в основном потоке кода.
  • Function Expression – объявление функции в контексте какого-либо выражения, например присваивания.
		var foo = function(параметры) {
			// тело функции
		};
		...
		foo(аргументы);

Immediately-Invoked Function Expression

Немедленно вызываемая функция в JavaScript (IIFE) — это синтаксическая конструкция, позволяющая вызвать функцию сразу же в месте ее определения.

		(function () {
			// ...
		})();

Первые скобки преобразуют код внутри себя в выражение, вторые скобки вызывают функцию которыя получается в результате выражения из первых скобок. Такие конструкции используются для создания локальных областей видимости переменных. Глобальный namespace остается 'чистым' т.к. переменные обьявленые внутри IIFE не видны вне функции.
